## How to use

[example](https://jscroot.github.io/croot.html)

html file :

```html
<script type="module" src="croot.js"></script>
```

js file :

```js
import { setInner } from "https://jscroot.github.io/element/croot.js";


setInner("demo","Dari croot.js");
```
